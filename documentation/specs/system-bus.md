# Ashet System Bus

## Pins available on the backplane

- D0 … D15 (Data bits)
- A1 … A23 (Address bits)
- BLS0, BLS1 (Byte selector)
- /OE (Output Enable)
- /WE (Write Enable)
- /ME (Module Enable)
- 4×GND
- 2×3V3
- 2×5V0
- IRQ

Total: 50 pins required, 64 pin connector used
Signal Pins: 45
